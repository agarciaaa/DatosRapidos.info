Aviso de Privacidad

Datos Rápidos - Aplicación Móvil Android

Versión 1.3.0 | Última actualización: Septiembre 2025

1. Información General

La aplicación Datos Rápidos es desarrollada y mantenida por Angel Garcia y Edgar Santana. Esta aplicación móvil está diseñada para permitir a los usuarios almacenar, gestionar y compartir información de forma rápida y segura en sus dispositivos Android.

Este aviso de privacidad describe cómo la aplicación maneja la información del usuario y establece las limitaciones de responsabilidad del desarrollador y la aplicación.

2. Recopilación y Almacenamiento de Información

2.1 Información del Usuario

  • Credenciales de acceso: Nombre de usuario y contraseña (almacenados localmente)
  • Datos personales: Notas, archivos, imágenes y cualquier información que el usuario decida guardar
  • Configuración de seguridad: Preguntas de seguridad y PIN de respaldo

2.2 Almacenamiento Local

Toda la información se almacena exclusivamente en el dispositivo del usuario utilizando la base de datos local Room Database. La aplicación NO envía datos a servidores externos ni comparte información con terceros.

2.3 Detección de Reinstalación (Nuevo en v1.3.0)

La aplicación incluye un sistema inteligente de detección de reinstalación que:

  • Detecta automáticamente cuando la aplicación ha sido desinstalada y reinstalada
  • Limpia automáticamente los datos de usuario anteriores para permitir un registro nuevo
  • Garantiza la privacidad eliminando datos residuales de instalaciones anteriores
  • No requiere intervención del usuario - funciona de forma transparente

3. Uso de la Información

La información almacenada en la aplicación se utiliza únicamente para:

  • Proporcionar las funcionalidades básicas de la aplicación
  • Permitir la autenticación del usuario
  • Gestionar y organizar los datos del usuario
  • Facilitar la búsqueda y recuperación de información

4. Seguridad de la Información

La aplicación implementa las siguientes medidas de seguridad:

  • Autenticación biométrica: Huella dactilar o reconocimiento facial
  • Cifrado local: Datos sensibles protegidos con criptografía
  • Base de datos segura: Room Database con migraciones automáticas
  • CredentialManager: Gestión segura de credenciales del sistema
⚠️ ADVERTENCIA IMPORTANTE

La aplicación Datos Rápidos NO garantiza la seguridad absoluta de la información almacenada. Aunque se implementan medidas de seguridad estándar de la industria, ningún sistema es completamente infalible.

El usuario reconoce y acepta que:

  • La información se almacena en su dispositivo personal
  • La seguridad depende también de la configuración de su dispositivo
  • Debe mantener su dispositivo actualizado y protegido
  • Es responsable de mantener sus credenciales seguras

5. Limitaciones de Responsabilidad

🚫 RENUNCIA DE RESPONSABILIDAD

EL DESARROLLADOR, LA APLICACIÓN Y CUALQUIER ENTIDAD RELACIONADA NO SE HACEN RESPONSABLES de la información guardada por el usuario, especialmente si se trata de información de carácter delicado, confidencial o sensible.

5.1 Información de Carácter Delicado

ESPECÍFICAMENTE SE ADVIERTE que la aplicación NO está diseñada para almacenar información altamente sensible como:

  • Información financiera (números de tarjetas, cuentas bancarias)
  • Documentos de identidad (pasaportes, licencias, números de seguridad social)
  • Información médica confidencial
  • Secretos comerciales o información corporativa sensible
  • Cualquier dato que requiera protección especial por ley

5.2 Uso Indebido de la Información

EL DESARROLLADOR NO SE HACE RESPONSABLE por el uso indebido, incorrecto o ilegal de la información que el usuario decida almacenar en la aplicación, incluyendo pero no limitado a:

  • Uso de la información para actividades ilegales
  • Compartir información confidencial sin autorización
  • Almacenamiento de información que viole derechos de terceros
  • Uso de la aplicación para fines no autorizados
  • Cualquier consecuencia derivada del mal uso de la información

5.3 Pérdida o Corrupción de Datos

El desarrollador no garantiza la integridad permanente de los datos almacenados. El usuario acepta que:

  • Los datos pueden perderse por fallos del dispositivo
  • La aplicación puede experimentar errores o bugs
  • Las actualizaciones del sistema pueden afectar la funcionalidad
  • Es responsabilidad del usuario hacer respaldos regulares

6. Responsabilidades del Usuario

Al utilizar la aplicación, el usuario acepta y reconoce que:

  • Es responsable de la información que almacena en la aplicación
  • Debe evaluar la sensibilidad de la información antes de guardarla
  • Es responsable de mantener la seguridad de su dispositivo
  • Debe cumplir con todas las leyes aplicables al usar la aplicación
  • No debe usar la aplicación para actividades ilegales o no autorizadas
  • Debe hacer respaldos regulares de su información importante

La aplicación se proporciona "TAL COMO ESTÁ" sin garantías de ningún tipo, expresas o implícitas, incluyendo pero no limitado a:

  • Lee y comprende completamente este aviso de privacidad
  • Acepta todas las limitaciones de responsabilidad establecidas
  • Reconoce que es responsable de la información que almacena
